# ITCA Kubernetes Workshop

## Prerequisites

You need to install next tools to be able to deploy the infrastructure and login to the Kubernetes cluster.

* [direnv](https://direnv.net/#basic-installation)
* [tfswitch](https://tfswitch.warrensbox.com/Install/)
* [kubectl](https://kubernetes.io/docs/tasks/tools/#kubectl)
* [helm](https://helm.sh/docs/intro/install/)
